  
│🌌 AetherChat:
A full-stack, real-time chat application featuring a glassmorphism dark-mode UI, secure authentication, and live presence tracking.

🌟 Features:

Secure Auth: Password hashing via bcrypt and JWT                          session tokens stored in secure HTTP-only cookies.
Real-Time Chat: Global chatroom and Private Messaging (DMs) powered by Socket.IO.
Live Indicators: Real-time online/offline presence status and typing indicators.
Persistent History: Chat logs stored in MongoDB and loaded dynamically.
Premium UI: Cosmic-purple glassmorphic design with responsive layout and dynamic avatars.

🛠️ Tech Stack

Frontend: React (Vite), Context API, Socket.IO Client, Lucide React
Backend: Node.js, Express, Socket.IO Server, JWT,       BCrypt
Database: MongoDB, Mongoose ODM
